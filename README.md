## Instrukcja uruchomienia

### Backend
1. Wejdź do katalogu `server`
2. Uruchom `npm install`
3. Uruchom `npm run dev` (serwer wstanie na porcie 3000)

### Frontend
1. Wejdź do katalogu `client`
2. Uruchom `npm install`
3. Uruchom `npm run dev`

### Uwagi
Projekt wymaga połączenia z bazą MongoDB (skonfigurowane w server/.env).
Zaimplementowano logowanie, rejestrację oraz mechanizm Refresh Token.
